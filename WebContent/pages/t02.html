需要引入的有三个 jquery.js、bootstrap.min.js 和 bootstrap.min.css

这里推荐使用百度的静态资源库(http://cdn.code.baidu.com/)上的Bootstrap资源。

百度的静态资源库的 CDN 服务，访问速度更快、加速效果更明显、没有速度和带宽限制、永久免费,引入代码如下：
新 Bootstrap 核心 CSS 文件
http://apps.bdimg.com/libs/bootstrap/3.3.0/css/bootstrap.min.css
可选的Bootstrap主题文件（一般不使用）
http://apps.bdimg.com/libs/bootstrap/3.3.0/css/bootstrap-theme.min.css
jQuery文件。务必在bootstrap.min.js 之前引入
http://apps.bdimg.com/libs/jquery/2.0.0/jquery.min.js
最新的 Bootstrap 核心 JavaScript 文件
http://apps.bdimg.com/libs/bootstrap/3.3.0/js/bootstrap.min.js
